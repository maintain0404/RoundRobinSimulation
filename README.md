RoundRobin 시뮬레이션
=================
RoundRobin 방식으로 스케줄링되는 운영체제가 돌아가는 것을 시뮬레이션한 간단한 프로그램입니다.

## 	실행환경
- goormIDE(Ubuntu 18.04 LTS)  
- GCC/G++ 9.1.0  

## 	실행 조건 - v1
- 10000회 퀀텀이 돌면 종료  
- 퀀텀 50ms  
- 자식 프로세스 10개  
- RoundRobin 방식  
- FIFO 실행 큐, 대기 큐 하나씩  
- 위 사항은 main_v1.c의 매크로로 정의되어 있

## 실행 조건 - v2
- RoundRobin 기반, 실행 큐 2개
- 우선 큐가 80%로 먼저 실행되고, 20% 혹은 우선큐에 없을 때 일반 큐 실행
